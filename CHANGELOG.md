# Changelog

All notable changes to the Blood Drip extension will be documented in this file.

## [0.0.1] - Initial Release

### Added
- Blood drip animation on error lines with red gradient effect
- Ethereal cursor movement trails with fade-out animation
- Spooky TODO icons (skull, ghost, tombstone) in the gutter
- Configuration options to enable/disable each effect
- Performance optimizations including animation pausing when editor loses focus
